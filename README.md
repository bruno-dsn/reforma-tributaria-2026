# Análise da Reforma Tributária 2026

Este projeto foi desenvolvido como parte do meu aprendizado em Análise de Dados, utilizando dados públicos reais sobre a Reforma Tributária brasileira com início em 2026.

## Problema
A Reforma Tributária altera significativamente a carga tributária sobre diferentes setores da economia. A complexidade da legislação dificulta a compreensão dos impactos reais para empresas e consumidores.

## Objetivo
Utilizar análise de dados para comparar alíquotas antes e depois da reforma e identificar os principais impactos setoriais.

## Fontes de Dados
- Emenda Constitucional nº 132/2023 – Senado Federal
- Lei Complementar nº 214/2025 – Diário Oficial da União
- Receita Federal do Brasil
- Ministério da Fazenda

## Ferramentas Utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab
- Looker Studio

## Estrutura do Projeto
- `data/`: dados utilizados
- `notebooks/`: análise exploratória
- `reports/`: relatórios e fontes
- `dashboard/`: visualização final
